Introduction to Ollama

Ollama is an open-source AI development platform that enables you to run, create, and manage AI applications locally. This documentation will help you get started and make the most of Ollama's features.

Note

This documentation assumes basic familiarity with AI concepts and development tools. If you're new to AI development, we recommend starting with our Quick Start Guide.

Key Features

Fast Development

Build and deploy AI applications quickly with our intuitive tools and frameworks.

Secure by Design

Enterprise-grade security with local-first architecture and data sovereignty.

Easy Integration

Seamlessly integrate with your existing tools and workflows.

Quick Start Guide

Installation 
# Linux/macOS
curl -fsSL https://ollama.ai/install.sh | sh

# Windows
winget install Ollama

Basic Usage

Python Example 
from ollama import Client

client = Client()
response = client.chat(model="llama2", messages=[
    {"role": "user", "content": "Hello, how are you?"}
])

print(response["message"]["content"])

Important

Make sure you have sufficient system resources available before running large models. Check our system requirements for more details.
